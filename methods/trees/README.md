# trees
Small trees used in methods development.

- big.tre
Balanced, fully bifurcating, 64 tips.

- felsentree.tre
The tree from Felsenstein's "worst case" (Felsenstein 1985, Fig. 5). 40 tips, two polytomous subclades of 20 tips each.

- w.tre
8 tips, two polytomous subclades of 4 tips each.

- x.tre
Balanced, fully bifurcating, 16 tips
